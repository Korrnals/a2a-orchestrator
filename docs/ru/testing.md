## Unit-тесты

278 тестов, покрывающих все модули:

- маршрутизация (R1–R6)
- управление сессиями
- паттерн «сага»
- подписи (Ed25519)
- WebSocket-стриминг
- поиск
- web-сервер (REST API)
- регистрация
- мультитенантная изоляция
- исправления безопасности (path traversal, timing-атаки, права файлов)

### Запуск

```bash
PYTHONPATH=src python3 -m pytest tests/ -q
```

### Только e2e

```bash
PYTHONPATH=src python3 -m pytest tests/e2e/ -q
```
